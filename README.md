# NewCool Consumidor - Derechos del Consumidor

Portal educativo sobre derechos del consumidor y proteccion SERNAC en Chile.

## Demo

[https://consumidor.newcool.io](https://consumidor.newcool.io)

## Caracteristicas

- Buscador de oficinas SERNAC
- Guia de derechos del consumidor
- Calculadora de garantia legal
- Proceso de reclamo paso a paso
- Informacion sobre retracto

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
